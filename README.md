#  Софтварный 3D рендер на javaFX
Это учебный проект по созданию 3D рендера на javaFX без стороннихх библиотек. 
Проект призван наглядно показать работу подобных программ "изнутри".  

## <ins> Интерфейс </ins>
Удобный и интуитивно понятное оконное приложение для работы с 3D моделями.  

## <ins> Рендеринг </ins>
Реализует задачу представления модели на экране и. 

## <ins> Математическая часть </ins>
Реализация математических операций, необходимых для работы с 3D моделями: 
+ Операции над векторами,
+ Афинные преобразования
+ Трансформация модели
+ Горячие клавиши и управление мышью
_______________________________________________________________________________________
  
<ins> Вектора </ins>
+ Vector - абстрактное представление вектора
+ AbstractVector - общие операции над векторами
+ Vector2D - операции над 2D векторами
+ Vector3D - операции над 3D векторами
+ Vector4D - операции над 4D векторами
  
<ins> Матрицы </ins>
+ Matrix - абстрактное представление матриц
+ AbstractMatrix - аналогично AbstractVector: для общих операций над матрицами (сложение, умножение, вычитание)
+ Matrix3x3 - операции над матрицами размера 3 на 3
+ Matrix4x4 - операции над матрицами размера 4 на 4
  
<ins> Прочее </ins>
+ LinearAlgebraEnigue - создание объектов
+ Camera - содержит матрицу проекций, направление камеры, вектор up
+ CameraController - управление движением модели с клавиатуры
+ CameraAdapter - управление движением можели с помощью мыши
+ LocalModel - содержит модель  в виде массива координат, операции над моделью
